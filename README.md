# arXiv_finder
arXiv の論文をまとめる.

## 概要
arXiv に投稿された論文の中で,

- Optimization
- Machine Learning
- Quantum Computer

関連の最新のものを取得し, 興味のあるものを集める.

## ローカル環境のセットアップ
仮想環境を構築する.

```sh
source setup.sh
```
